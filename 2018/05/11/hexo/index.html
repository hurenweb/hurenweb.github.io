hexo教程

发表于 2018-05-11

hexo安装和指令

  1. 当然得先安装nodejs/npm/cnpm/github
  2. 打开官网 查看基本使用指令
  3. 全局安装cli:npm install hexo-cli -g
  4. 初始化下载博客:hexo init blog
  5. 安装npm依赖包:cd blog,npm install
  6. 开启服务:hexo server或者hexo s
  7. 打开浏览器http://localhost:4000/
  8. 修改后清除缓存:hexo clean
  9. 退出服务:ctrl+c,两次后确定
  10. 新建文件：$ hexo new "My New Post"
  11. md文件目录:D:\www\blog\source\_posts
  12. 生成静态文件:$ hexo g
  13. 部署到远程站点:$ hexo d

next主题设置

  1. 官网查看基本使用方法
  2. 官方文档
  3. 使用git下载主题：文件存放目录themes/next,如果目录不存在需要提前创建， $ git clone https://github.com/iissnan/hexo-theme-next themes/next,不想使用git指令也可以直接下载源代码，放到对应目录
  4. 设置主题：打开配置文件blog/_config.yml，language: zh-Hans , theme: next
  5. 配置主题：打开主题配置文件blog\themes\next/_config.yml，设置为单栏紧凑型主题scheme: Mist，清空缓存重启服务，浏览器查看效果。
  6. 设置首页为文章摘录形式，默认为显示全文，auto_excerpt:true
